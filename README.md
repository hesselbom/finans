finans
======

Simple CLI to list a stock using the Yahoo Finance api

Example usage:

`finans goog`

Would give you the following result Nov 12, 2014:

    ┌──────────┬────────┬─────────────┬────────┬────────┐
    │ Exchange │ Symbol │ Name        │ Price  │ Change │
    ├──────────┼────────┼─────────────┼────────┼────────┤
    │ NasdaqNM │ GOOG   │ Google Inc. │ 547.58 │ -0.51% │
    └──────────┴────────┴─────────────┴────────┴────────┘

Enjoy!
